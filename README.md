# King John 1205: A Royal Chronicle

A text-based historical adventure game where you play as King John of England throughout the pivotal year of 1205.

## Overview

Experience a year in the life of one of England's most controversial monarchs. Navigate political intrigue, military crises, baronial tensions, and church conflicts while managing resources and relationships. Your decisions will shape history—will you follow the historical path, or forge a new destiny?

## Features

- **Day-by-day simulation** of the entire year 1205 (365 days)
- **Resource management**: Treasury, royal authority, military readiness, papal relations
- **Relationship system**: Track loyalty of 8 key barons
- **Regional stability**: Manage 6 regions across England and the continent
- **Historical events**: 60+ major historical moments
- **Multiple endings**: Based on your decisions and resource management
- **Narrative branching**: Your choices create alternate history paths

## Historical Context

1205 was a crucial year for King John:
- Lost Normandy to France in 1204, earning the nickname "Softsword"
- Planned invasion to reclaim French territories (cancelled due to baronial refusal)
- Death of Archbishop Hubert Walter triggered church crisis
- Constant movement across England, issuing writs and charters
- Military preparations and financial pressures
- Seeds of baronial discontent that would lead to Magna Carta (1215)

## Installation

### Requirements
- Python 3.9 or higher
- No external dependencies required (runs on Python standard library)

### Setup
```bash
# Clone the repository
git clone <repository-url>
cd King_John_Game

# (Optional) Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# (Optional) Install optional dependencies
pip install -r requirements.txt

# Run the game
python main.py
```

## How to Play

1. Start a new game or continue from a save
2. Each day, you'll receive reports and face decisions
3. Make choices that affect resources, relationships, and the kingdom
4. Navigate through the year, managing crises and opportunities
5. Reach December 31st to see your final score and ending

### Controls
- Use number keys to select choices
- Type commands when prompted
- Save anytime during gameplay
- Multiple save slots available
